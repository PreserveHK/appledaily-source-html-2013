糯米鴛鴦　叠住上

糯米鴛鴦　叠住上

無論中國人或日本人，過年時，都會用糯米來製成糕點，實際作用在於寒冬中暖暖胃；而且亦具有一些吉祥的象徵意義，如我們春節的發糕表示「發財、開運」，年糕寓意「年年高升、四季平安」。今日我們不如試試來到米芝蓮食府，學弄內外兼備的得獎菜式鴛鴦糯米飯，用作招呼親朋定不失禮。
記者：張文蕙　攝影：伍慶泉

糯米有白、黑、紅之分，營養十分豐富，含有蛋白質、脂肪、糖類（以澱粉為主）、鈣、磷、鐵、維他命B1、維生索B2等。
白糯米又分為粳糯及秈糯，兩者外觀皆呈白色而不透明，粳糯米粒較短且圓，秈糯米粒則較細長。米粒較短的粳糯，人們亦多稱之為圓糯，適合釀酒、製湯圓及紅粿等甜點。而細長形的秈糯，人稱長糯，則多用作包糭子、作米糕、油飯等鹹點。糯米既可以製酒，故又被稱為酒米。
至於黑糯米又稱紫米，是糙米的一種。煮熟後的黑糯米香味極佳，色澤呈紫紅色，亦含有一般糯米的特性，營養價值極高，可做壽司、甜米糕、甜品，或與白糯米混合包糭子。最後一種紅糯米，原稱為「紅粟米」、「黑粟米」，色澤呈淡紅色，是台灣阿美族人傳統的產物。
黑糯米煮前不需要浸，但一定要加調味，一般做法是加入甜味，用來做甜品，這樣就可以去除當中的苦澀味。煮法方面很簡單，只要像一般煲米飯就可以了。不過，白糯米就需要先浸一晚，然後隔水蒸熟，才能煮成煙韌的糯米口感。

鴛鴦糯米飯

這個鴛鴦糯米飯，是創意之作。由兩種不同顏色的糯米混合煮成，叠擺得猶如藝術品，吃起來更令人驚訝地發現，原來甜甜的荔茸糯米球與惹味的煎糯米雞，原來都可以如此配襯！

難度：★★★★　時間：1小時30分鐘　份量：2人

材料：
白糯米/黑糯米/荔茸各4両、紅糙米/雞肉（切粒）/提子乾/雜菌各1両、鹹蛋黃1隻、冬菇（切粒）2隻、雞湯/菇菌湯各2両、菜湯1湯匙、糯米酒1茶匙、砂糖半茶匙、鹽/胡椒粉/麻油/生粉各少許、春卷皮1片、紅蘿蔔條1條

做法：
1. 白糯米及紅糙米分別煮熟，各加入鹽、胡椒粉、麻油拌勻。

2. 將（1）混合後，包裹已炒熟的雞肉、冬菇及蒸熟了的鹹蛋黃，成長方狀，然後整個煎香備用。

3. 將黑糯米加水煲成飯，加入提子乾、雜菌、雞湯、糯米酒後搓成圓球備用。

4. 將荔茸包裹（3）成圓球，以滾油炸至表面起絲。

5. 菇菌湯加入鹽、砂糖、胡椒粉蒸熱1分鐘，加入生粉成芡汁上碟。依次放上（2）、（4），頂端加上炸好的春卷皮卷及紅蘿蔔條即成。

小貼示：
炸荔茸球時先放少許落油鑊，確定溫度足以炸至起絲，才整個放下去，以免浪費。

狀元煎茶粿

茶粿有鹹甜兩種吃法，鹹的通常以眉豆、花生作餡料。今次師傅以菜甫、冬菇、蝦米等香口的配料，製成茶粿後更將其底部煎香，上面煙韌、底部香脆！難怪少年食神Michael Lam每次來都要點這個！

難度：★★★　時間：45分鐘　份量：2人

材料：
外皮：糯米粉10両、熟澄麵6両、粘米粉3両、砂糖3両、清水5両、豬油4両

餡料：雞肉/豬肉各3両、菜甫/冬菇/韭菜各2両、花生（已去殼去衣）1.5両、蝦米/中芹各1両、鹽/胡椒粉/杞子各少許、

做法：
1. 先做茶粿皮，將糯米粉、粘米粉及砂糖加入清水搓勻成粉糰，再搓入豬油、熟澄麵製成皮料，其間起粒時便加入適量清水，完成後以保鮮紙包好，放於雪櫃備用。

2. 跟着預備餡料，先將豬肉、雞肉、菜甫、蝦米、冬菇、中芹及韭菜切成幼粒，然後，連同花生，加入鹽及胡椒粉炒勻。

3. 取20克皮料壓成塊狀，包入15克餡料。

4. 以一條韭菜包裹（4），以杞子點綴，蒸8分鐘。

5. 將茶粿底部煎至金黃即成。

椰絲奶黃湯圓

快過年了，家家戶戶都愛弄湯圓喻意團團圓圓，當你弄芝麻他又弄花生時，卻有家人想起流沙奶黃都很可口，可惜奶黃卻不是人人懂弄得美味，黃師傅今次就公開美味奶黃的秘方，只要蒸煮時間不過長，咬下去奶黃香軟，在家同樣能熱辣辣地弄出流沙奶黃湯圓！

難度：★★★　時間：35分鐘　份量：2人

材料：
糯米粉半斤、豬油1両、砂糖/熟澄麵各2両、水/奶黃/椰絲各適量

奶黃：雞蛋5隻、糖5両、麵粉/奶粉/栗粉各5錢、吉士粉1.8錢、椰汁/花奶各8両、牛油2両、保鮮奶10両

做法：
1. 將糯米粉、熟澄麵、豬油、糖搓勻，其間起粒時便加入適量清水，完成後以保鮮紙包好，放於雪櫃備用。

2. 將奶黃材料混合後蒸熟，並搓分成數顆圓球。

3. 以4份麵粉，對1份奶黃的比例，將麵粉包裹奶黃。

4. 將（3）蒸熟後，於表面撒上椰絲即成。
