港人熱捧　Model S下季落地
Tesla駛入中國　撼比亞迪

【財經專題】

在美國銷情火爆的電動跑車Tesla最新型號Model S，最快今年冬季抵港。Tesla亞洲區企業傳訊主管土肥亞都子（Atsuko Doi）接受專訪時透露，位於北京的全球最大陳列室即將開業。隨着Tesla進軍中國，國產比亞迪（1211）將面臨最大挑戰。
記者：鄭柏齡

美國Tesla在2010年已進軍香港。土肥透露，公司首款電動跑車Roadster在港大賣，「全球賣出2,500部，香港已佔其中50部」，不少名人如趙式芝、馬清揚皆為捧場客。有見及此，去年10月Tesla將最新款、美國大賣的電動跑車Model S引入香港，供車迷試車及預訂。Tesla香港區銷售經理呂建韜指，「係亞洲區第一部，半年內試咗大概1,000次，高𥧌期一日有16人試車」，他又指，「就算預約試車，都要排兩個星期隊」。

香港售75至100萬元

公司未公佈Model S香港車價，但呂建韜指，參考歐洲訂價，介乎75萬至100萬元不等。他不肯披露預訂情況，但表示「嚟試車嘅，大概有四分一人買」，推算計，Model S已接逾250張定單；他補充車主多為港人，亦有在港工作的內地Banker、專業人士等，最快今年第四季交貨。
反觀對手國產比亞迪，5月時引入兩款電動車e6及e6 Premier，前者出租作的士，後者為私家車版。代理商森那美透露，售價44.8萬元的私家車版仍「未發市」，「公司想等充電站風波過咗後，靜番啲，先再搞推廣活動」；電動的士方面，現時共有14部在港行駛，不及公司早前預期、年底前達45部的三分之一。
同樣是電動車，一架「來佬貨」，一架「國產貨」，Tesla Model S及比亞迪e6在港銷情可謂天與地。記者親身體驗兩架「電車」，前者跑味十足，但價格高昂，可算是「有錢人玩意」；比亞迪e6價格相宜，但採用中國充電制式，需自建充電站較麻煩。

Model S跑力勁　e6較平

Model S最大特色是充滿科技感，跟傳統汽車不同的是，Model S車頭設有一部恍如iPad的「個人電腦」，所有裝置如天窗、冷氣，以至汽車性能，統統可透過touch screen調整；加上接通互聯網，可以像智能電話般，隨時在線上更新跑車功能，毋須送回維修中心。
電動車賣點是環保及寧靜，Model S跑力更為驚人，輕踏油門，0至100km/h僅約4秒，連記者亦被其「推背力」嚇了一跳。呂建韜指，與比亞迪的鐵電池不同，Model S採用較普遍的鋰電池，可於全港數百個政府充電站叉電。
國產比亞迪e6售價約44萬元，比Model S平近半。不過，論外形、性能、車內裝置，都比Model S「差一截」。此外，比亞迪採用中國充電制式，與政府充電站「相沖」，車主買車後，要自行透過代理，在停車場加裝充電站。
股價方面，Tesla亦拋棄比亞迪幾條街，前者今年飆升3.5倍，市值增至1,449億元，後者今年則升35%，連同A股，市值約950億元。
