遊搜好行：杜拜無牌飛車　Ready Go!

杜拜賽車國際聞名，引擎咆哮聲劃破天際直達無邊沙漠，踩油狂奔、高速入彎、甩尾飄移……要落場體驗？No Problem！無車牌都得，坐進高卡車，Ready Go！

記者、攝影：符樂

有車牌也輸了

要在杜拜體驗刺激的速度快感，除了在沙漠馳騁還可回歸賽車場。在Dubai Autodrome，無論是一級方程式賽車（Formula Car）、電單車或以房車改裝的賽車往（Touring Car）都同樣有得玩，更有Audi TT、Subaru STI任君選擇。但我不是城城，Audi R8輪不到我高攀，還是轉投入門級、時速不超過60km的高卡車懷抱！雖然是入門，但過程相當認真，連身賽車衣、頭盔及手套一絲不苟，笨拙如我都頓變一級賽車手，人靠衣裝果真沒錯。
Kartdrome的賽道全長1.2公里，可同時容納17人作賽；扁小的高卡車操作簡單，只有油門與煞掣，沒有駕駛執照亦可參與。聽導師講解後便可動身出發，賽事中集齊中東、日本、歐美或來自不同地方的選手，離開Pit區後個個都踩盡油門狂奔，尖銳的引擎聲此起彼落。早已「甩P」的我恃着車牌在手，以為大有優勢，殊不知駕駛私家車與駕駛高卡車是兩碼子的事，每當轉彎時軚盤一扭，即失控打白鴿轉。偷窺一下從後飆車而至的外國人，原來轉彎扭軚的幅度不能太大，扭一點再回一點軚就夠了。恍然大悟，再來一次飄移，誰料轉頭我已失控撞上車胎陣。我承認，是次作賽成為包尾大王的我丟盡港人面子，撞上車胎防護欄的一刻更讓頸項疼痛不已。然而前無障礙的飛馳，其快感好比減壓靈藥，更重要的是：賽後握握手，識到朋友使乜愁！

商場半天吊

一場賽車未夠喉，再來到杜拜「時代廣場」離地上天挑戰膽量！位於商場的歷奇公園，玩家須走畢離地約八米高的重重機關：木柱、木塊、車胎、繩網看似簡單，但職員透露，肌男羅仲謙早前到來竟嚇得臉青青，耍手擰頭怪叫連連，蹉跎良久才成功過關。我在身上只繫了一條安全帶，以及腳下安全網欠奉的情況下，雙腿也不禁發抖，體力、平衡力、膽量、專注力瞬間推至極點！一步一驚心跨過直徑僅十厘米的木頭後，稍作歇息環顧四周，不少在場的遊人均報以好奇的眼光甚至搭訕。站在商場二樓接近天花板的位置，與陌生人聊天，還真是奇趣的經歷。

踏足酋長飯堂

在杜拜高樓燈飾的映襯下，晚上來一頓中東菜乃是上佳配搭。中東菜以黎巴嫩菜為首，曾獲《Time Out》選為最佳中東菜餐廳的Al Nafoorah，其Sidon Set Menu解決了我的選擇障礙症：開胃的醃製珍珠瓜、紅菜頭、蒜頭配各式豆類、薯仔、芝士是前菜；主菜是烤石斑、羊肉及大蝦，連同甜品共超過20道菜。面對一桌美食請保留一點矜持，這裏是杜拜酋長的飯堂，也許有機會碰見酋長的石油王朋友啊！
