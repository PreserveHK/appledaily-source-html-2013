私駕路：西沙避人潮 雙營

香港拍拖勝地早就被遊客攻陷！情人節行商場？咪搞！去離島？長洲可能比中環更熱鬧。連貝澳及灣仔營地都被霸佔，想避人潮，不如駕車往西貢西沙路，水浪窩有個觀星台，有燒烤場，近在咫尺又有營地。這種「港式」營地，旅遊巴泊車困難，造就私家車主們的隱密地。今夜，我們雙雙去觀星﹗


記者︰徐影珊　攝影︰蔡家輝

模特兒︰Hidy Yu @ GME、William Lau

水浪窩這名字，經常與麥理浩徑齊齊出現，因為它是麥徑第三段的終點、第四段的起點。我們已相識六載，今年就由我為你去準備節目吧。知道你喜歡駕着愛驅Audi TT Coupe周圍去，我為你設計了一條以公路為主的路線，從九龍入西沙路，經由隧道過吐露港往馬鞍山，全程行高速，駕車必定暢快！然後，我們可在西沙路的水浪窩路邊停車場把車泊好，往觀星台步行，只不過數十級樓梯就到。時間尚早，我們可沿樹林研習徑走到麥理浩山徑第四段，登上馬鞍山郊野公園去，可遠望整個西貢「平原」呢﹗

行20分鐘到營地

下午Tea Time ，我喜歡西沙茶座，而你就鍾情大網仔巴士站外的茶座，不如你決定吧。我們熱愛大自然，難得遠離繁囂，能把步伐慢下來真好，冬季日落時間早，我們可駕車回到水浪窩附近路邊停車場，拿取露營用品，往漁農署指定的水浪窩營地去，約步行20分鐘，不算辛苦。我會自備食物做 BBQ及Cook Set 煮食。露營的地方有流動廁所，泊車地點不太遠，靜悄悄的可避開自由行，今晚大概會很浪漫吧，希望能見到滿天星星許個願。

行程路線

11:00am→九龍出發，經吐露港往馬鞍山，入西沙路
12:00nn→抵達水浪窩路邊停車場
12:30pm→開始行大運，沿樹林研習徑往麥理浩第四段上馬鞍山郊野公園
3:15pm→開車返回西沙路邊茶座，或往大網仔方向駕車，抵達利昌或有利士多
5:00pm→返回水浪窩路邊停車場泊車
5:30pm→沿麥理浩徑第四段入露營營地
7:00pm→BBQ或煮晚餐，觀星，度過甜蜜情人節

型格情侶跑車

我們最喜歡有私人空間，Audi TT Coupe是上選，前座空間寬闊，後座則通常用來放置手袋及用品，最適合我們二人周圍去。男友就最愛它的跑車式車身設計，此版本加有S Line包圍，配合17吋合金輪圈及可調校升起的尾翼，他說夠型格。採用1.8 公升TFSI 引擎，最大馬力160匹，最大扭力250Nm，七前速S tronic波箱，配合加減波及S Mode，男友說，在公路飛馳時感覺很暢快！
