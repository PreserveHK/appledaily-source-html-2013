西九戲棚月尾開鑼　
設計恍如宮殿廟宇

【本報訊】西九大戲棚本月30日開鑼，今年繼續以傳統竹棚搭建，並由建築師林偉而負責空間概念設計，令市民恍如置身古代宮殿及廟宇。西九管理局今年又推出隱藏AR code的大花牌，市民到場後利用智能手機及平板app對應，便可觀看日後戲曲中心的立體模型。

花牌隱藏AR code

曾於前年為維園綵燈會設計巨型魚形燈籠的林偉而表示，大戲棚的規模是全港最大的戲棚，為凸顯其傳統外貌風格，今年特別減少在外掛上巨型花牌，並為鋅片外加上橙黃色，遠看時更似傳統的中國宮殿廟宇，糅合當代藝術及傳統文化。
西九管理局表演藝術行政總監茹國烈指，今年各場收費粵劇及當代音樂表演大部份場次的門票已售出七、八成，部份大老倌擔綱的劇目更已售罄。他又指，即使市民對劇目無興趣，也可到戲棚外的市集閒逛，26個檔攤會出售傳統手工藝如糖公仔等，令現場有墟市的氣氛。管理局今年更在巨型花牌隱藏AR code，只需使用智能手機或平板電腦下載官方app，對準花牌掃描便可在屏幕見到日後戲曲中心立體模型，做到新、舊互動效果。
