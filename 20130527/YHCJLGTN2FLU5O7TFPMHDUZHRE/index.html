蘋機會：走出平板　新Surface Pro打開天窗

這片藍天，是否像你我都曾經用過的Windows牆紙？直到RT版出現，才被鋪到密麻麻的磚塊所取代！最新Surface Pro這次唔再齋篤芒，系統提升為擴展力更強的Windows 8專業版，真正用到MS Word、Photoshop等桌面版軟件。
記者：陳國輝　攝影：許有達

模特兒：Irene@Zmodel

內置Win8 Pro做「真電腦」

前朝Surface RT，或許新鮮感大於實際用途！例如它夠輕巧，薄身像iPad，然而卻只能安裝32Bit Windows RT，硬件也僅提供流動平板專用的1.3GHz四核心處理器，不過最大問題是，軟件只能使用很簡單的Apps，想學筆電用複雜的文書處理，甚至開Photoshop執相，絕對無可能！
至於Surface Pro，似乎正為解決此問題而出現。它內置筆電的硬件規格，包括採用Intel Core i5 1.7GHz處理器、4GB記憶體，更重要是可安裝64Bit Windows 8專業版及配置64GB和128GB硬碟，無論效能、擴展性也明顯比上代勁多多聲！當然，進化成真正電腦，代價都唔少，厚度方面佢就接近14mm，重差唔多1kg，加埋塊Type Cover還不止；同時也肯定比平板更食電，有網站測試發現，電池僅足夠連續使用約兩小時，拎出街用，那隨時連一套電影都捱唔住！

附送觸控筆　隨畫隨寫

是否喜歡將Surface Pro結合Type Cover，扮成好似真電腦咁使用，純粹個人喜好！當然，此機本身也支援10點輕觸式操作，全程齋篤芒都搞得掂。例如使用MS Word，你可隨時於畫面下方Call塊虛擬鍵盤出來，甚至打機時，更可在機身加轉駁器使用XB360手掣，成部Console上身咁；但個人覺得更好用是，跟機會附送一支手寫筆，除了可直接書寫文字、繪畫，當你上網瀏覽網頁，更可控制頁面上下捲動，好過用手指篤到污糟又多指紋吧！
