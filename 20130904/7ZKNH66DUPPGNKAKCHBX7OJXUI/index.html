立會報告擬落重藥轟湯顯明

【本報訊】立法會政府帳目委員會跟進審計報告有關廉署倡廉教育章節，五場公開聆訊均針對前廉政專員湯顯明任內涉失當的行為。據了解，立法會秘書處已草擬有關報告的初稿，對湯任內以公帑送贈厚禮予內地高官、頻頻外訪、飯局超支及貯備大量名貴茅台餐酒等多宗罪表示關注。委員會將於本月16日舉行閉門會議，商討加入不同程度的批評字眼、以及應否點名批評湯顯明。據悉委員傾向「落重藥」批評湯。

報告將於10月完成

立法會政府帳目委員會早前已完成聆訊，聽取前廉政專員湯顯明及現專員白韞六等的供詞；委員會主席石禮謙日前稱報告已完成初稿，將於10月完成。據了解，初稿主要提及三大方向，包括指出湯顯明任內五年不必要地購入大量酒品；外訪次數過於頻繁，以公帑宴客超支；以公帑向不同單位送贈厚禮等。
另外，報告亦提及廉政專員的法定職責是專注於本地的肅貪倡廉工作，但審計報告揭露的超支問題，頻繁與內地官員交流，均反映湯顯明的行為超出其法定職責。委員會將於本月16日舉行閉門會議，商討報告關鍵字眼及是否點名批評湯顯明。據悉委員傾向落重藥予以批評。由於委員會不想影響廉署針對湯顯明的刑事調查，故亦需要諮詢法律意見，研究何時適宜公開報告。
另外，廉政公署公務酬酢、餽贈及外訪規管制度和程序獨立檢討委員會主席周松崗指，委員會在過去四個月召開16次會議，並向特首提交報告，但由於政府需要剔除部份調查廉署的內容，相信一、兩個星期後才可公開報告。
