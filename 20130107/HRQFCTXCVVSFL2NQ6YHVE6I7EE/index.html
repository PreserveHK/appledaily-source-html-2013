味噌增味　素食補品

日本傳統的味噌是以大豆煮熟攪碎搓揉後，加入海鹽調味，搓勻後放入米麴或麥麴，最後放入木桶內封存發酵，兩年後味噌成熟，除去表面一層發霉的麵豉便成。但時至今日，很少人會自己親手做，市面亦已有很多不同口味的味噌供選擇。今日請來日航酒店的苗加師傅，以不同的麵豉調配各種惹味菜式。
記者：張文蕙　攝影：蔡家輝

示範：苗加昌彥

（香港日航酒店Sagano總廚）
擁有三十年入廚經驗，先後受聘於京都多間五星級酒店，擅長正宗懷石料理，愛烹調考刀功的菜式，因此其菜餚都有如藝術品，色香味俱全。

味噌是從黃豆、鹽及不同的種麴發酵而成，又叫「麵醬」或「麵豉醬」（miso）。從西方營養學角度看，味噌（日本麵豉醬）是植物食品中少數含有維他命B12的，可防治貧血；素食者特別易有B12不足的情況，味噌就是方便又美味的「補品」。味噌有豐富的蛋白質，經過發酵已變成胺基酸，很容易被身體消化吸收，又由於它不經高溫消毒，當中有大量活性益菌，跟乳酪相似，怕乳酪寒涼，不妨改用味噌湯代替。此外味噌更可使血液保持微鹼性，有中和血液中過量酸性物質的作用。味噌的煮法，其實很簡單，為保持其中的活性益菌，烹調盡量不要超過3分鐘。
味噌的種類繁多，大致上可分為米麴製成的「米味噌」、麥麴製成的「麥味噌」、豆麴製成的「豆味噌」等。若以口味來區別，則可略分為「辛口味噌」及「甘口味噌」兩種，前者比較鹹，後者則較甜、較淡。這種口味上的差異是因為原料比例不同所造成，通常麴的比例較重的，成品偏向甘口；若是鹽的比例較重，成品就偏向辛口。頗具名氣的「信州味噌」，是辛口味噌的代表；而關西的白味噌及九州味噌，都是頗具代表性的甘口味噌。味噌顏色的深淺主要是受製麴時間的影響，製麴時間短，顏色就淡，時間越長，顏色也就變深。

八爪魚麵豉酸食

八爪魚十分鮮味，而且啖啖肉！今次苗加師傅就利用這種煙韌的海鮮配合清甜的白味噌一起烹調。白味噌味道較清，不太搶味，很適合用作配搭鮮味的食材。師傅為令菜式更惹味，更在白味噌中加入醋及辛子，這個八爪魚麵豉酸食十分開胃，作為餐前小點真不錯！

時間：20分鐘　難度：★★★　人數：2人

材料：
大八爪魚腳2條、白味噌100克、日本豉油50毫升、葱適量、醋50毫升、砂糖40克、辛子15克、薑芽數片、幼鹽適量、鹽少許

做法：
1. 八爪魚放入加了豉油和幼鹽的水中煮沸，熟八爪魚放涼後切片。

2. 在另一個鍋中將葱以加鹽的滾水灼熟後切段。

3. 將白味噌、醋、砂糖、辛子置於盤中攪拌成醬汁。

4. 將薑芽切絲後沖水。

5. 將醬汁置於碟上，再放上葱及八爪魚片，最後以薑芽伴吃即成。

麵豉煮鯖魚

鯖魚是一種高蛋白、低脂肪又易被人體吸收的食物，內有二十碳五烯酸（EPA）與二十二碳六烯酸（DHA），EPA具有擴張血管、防止血液凝結等作用；而DHA對大腦細胞更有增生作用。今日師傅以這種營養特高的日本魚特產，配合味道較淡的田舍麵豉和較濃味的紅麵豉煮成醬汁，魚腥味全消，十分惹味。

時間：30分鐘　難度：★★★★　人數：2人

材料：
鯖魚1條、清酒30毫升、水300毫升、砂糖50克、薑5片、梅乾1粒、田舍麵豉30克、紅麵豉30克、葱碎/香葉各少許

做法：
1. 鯖魚開邊分成3份，將中骨取去，用鉗子取去細碎的骨。

2. 起皮後，將皮灼熟，然後放在冰水中令魚皮變得爽口。

3. 將清酒、水、砂糖、薑片及梅乾煮沸後，放入魚肉，水滾後以慢火煮5分鐘，加入田舍麵豉及紅麵豉煮約10分鐘。

4. 將魚肉及醬汁上碟，伴以冰鎮好的魚皮、葱碎及香葉即成。

日式麵豉蘿蔔盒

秋冬當造的蘿蔔，一般用來燉煮，苗加師傅卻想出將蘿蔔化成美麗的盒子，中間就放着他精心調配的麵豉醬料，看上去已令人心動，吃起來更是每一口都有醬料，十分有心思！蘿蔔用昆布水煮過，變得更鮮甜。不過這道菜最難在於切出美麗盒子的模樣，大家宴客前要多多練習了！

時間：25分鐘　難度：★★★　人數：2人

材料：
日本蘿蔔1個、柚子茸適量、昆布數片、水適量

醬料：白麵豉100克、酒40毫升、味醂10毫升、砂糖40克

做法：
1. 將醬料材料以慢火煮，其間需攪拌以免黏鍋底，關火後加入柚子茸備用

2. 日本蘿蔔去皮，切出約7cm厚件，再切成六角形。在中間部份以細匙刮出一個小孔。

3. 切出蘿蔔頭，留作「盒蓋」，將蘿蔔葉切成玫瑰狀。

4. 將蘿蔔放入昆布水中煮至腍身。

5. 將醬汁放入蘿蔔的小孔中，並伴以蘿蔔「盒蓋」，即成。

燒麵豉茄子

日式茄子的煮法不是天婦羅就是原條燒，苗加師傅今次將茄子炸過後再燒，這樣油份也減少很多。他用兩種濃淡不一的麵豉，鴛鴦色地厚厚塗在茄子表面，這時你終於能真切地品嚐到，原來不同麵豉的甘甜與鹹香惹味是如此刺激味蕾的！

時間：20分鐘　難度：★★　人數：1人

材料：
茄子1個、油適量

白麵豉醬：白麵豉100克、酒40毫升、味醂10毫升、砂糖40克

紅麵豉醬：紅麵豉100克、酒40毫升、味醂20毫升、砂糖40克

做法：
1. 茄子切成每件約5cm厚，以針刺上小孔，放入滾油中炸至軟身。

2. 將炸好的茄子，放在燒爐燒香。

3. 白麵豉醬及紅麵豉醬分別以慢火煮後，塗於茄子表面，再燒約2分鐘即成。
