雲想衣裳：闊衣加長裙　懶佬變型

很多人都說做靚女，就不要做懶女人；但我認為兩者是可以並存！好似經典Loafer在春夏季將繼續大放異彩，既悠閒、有型又易穿。所以話，要靚又要懶點會無可能！

常見四種類型Loafer

1. Venetian Style Loafer：設計簡約，鞋面無任何裝飾的淺開口Loafer。

2. Penny Loafer：又稱Weejuns，在鞋面上方跨過一條帶子。

3. Tassel Loafer：是在鞋面上繫上流蘇。

4. Gucci Loafer：即是指Gucci在鞋上的經典金屬扣環設計，更有作家認為作為進入上流社會最基本的身份標籤。

隨心率性的鞋

別以為着Loafer就要配Casual look，其實Loafer鞋款眾多，變化度和可塑性都很高，在襯衣配搭上絕對可襯出不同風格！其實Loafer剛流行時，還可稱為Slip-on，隨着不同設計的出現，傳到美國便稱為Loafer，亦有隨心率性的意思。要數最經典的造型莫過於Penny Loafer，充滿歐陸的紳士氣派，在歐美的潮流中從未埋沒過。Penny Loafer在1950年在美國校園裏被學生們在鞋頭帶子上塞入一便士（Penny）作表徵而正式流行；1970年更成為美國商人的身份象徵；1980年，已故歌手Michael Jackson穿着Penny Loafer大跳Moon Walk而開始流行全世界。

Style 1：韓妹Street Style

皮褲或皮裙和大碼上衣如冷衫和衞衣，都是韓妹的必備Item；剛及腳踝的長裙已夠長，所以不一定襯短靴才能拉長身形；反之配上窩釘、絲絨或刺繡圖案平底鞋，同樣夠亮眼。另外，雖然小腿較粗和短的女孩穿平底鞋可能會自暴其短，但長裙可掩飾這個缺點，平底鞋一樣大派用場。

Style 2：斯文學院派

Loafer一向熟口熟面，皆因學生年代總會見到它們的蹤影；加埋永恒不死的豹紋流蘇Loafer，可以說是學院派的變種。

Style 3：型格Rock妹

Loafer是紳士鞋的變種，襯上西裝外套自然型格十足。用來搭配九分褲或Skinny Jeans更是不死的組合，但如果你認為牛仔褲太單調，大可穿上印花Print或有色Skinny Jeans，束腳和吊腳褲都可以。加上春夏回歸Glam Rock潮，Metallic尖頭Loafer絕對是Rock妹之選。

Style 4：日系少女風

雖然黑色流蘇Loafer比較中性，但學習日本妹玩層次，穿上一對短襪，提高顏色對比度，以柔制剛；同時也不怕俾人話着鞋唔着襪，老套兼核突！不過提醒大家，Loafer鞋的精髓就是露出腳眼，所以一係就唔着襪或着腳眼襪，一係就好似模特兒咁着短襪玩撞色，同樣Fashionable！

歌劇變奏

不可不知道的是Loafer也是宮廷歌劇鞋的變奏，這對長青便鞋以前是在正裝場合才會穿着的。而因為秋冬流行一股宮廷復古熱，各品牌都加入秀麗的刺繡和天鵝絨鞋面，Christian Louboutin與Jimmy Choo同樣推出有如宮廷鞋款的華麗變奏，更加入窩釘元素甚或好似Alexander McQueen推出一系列蛇皮及骷髏印花，Rock味甚濃。厚底Loafer鞋款也是秋冬季主打之一。若想增添女人味，鞋面的水晶裝飾、金絲緹花裝飾、蝴蝶結配鮮色緞面都是春夏的入流配搭。
