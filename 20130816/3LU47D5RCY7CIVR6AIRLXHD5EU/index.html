三星下月推智能手錶

【本報綜合報道】蘋果與三星下月再掀惡鬥！報道指三星為爭飲「頭啖湯」，將於9月4日在德國搶閘發佈Galaxy Note 3和智能手錶Galaxy Gear，務求要快過宿敵蘋果推出新機。

搶先蘋果出新機

英國科技網站3g.co.uk昨日報道，三星Galaxy Note 3將於下月4日在德國柏林消費電子展登場。三星亦有意將智能手錶Galaxy Gear一併推出。報道指，Galaxy Note 3的外型與Note 2略有不同。Note 3機身兩邊的邊框或較窄，但仍會維持配備S Pen設計。新機屏幕將擴大至5.68吋，而且屏幕與Galaxy S4一樣，可懸浮觸控。Note 3將採用8核心處理器，並配備3GB內置記憶體。用戶將有16GB、32GB或64GB三種記憶容量選擇。
較早前有報道指，蘋果公司計劃於9月10日為新iPhone舉行大型發佈會，預料iPhone 5S和廉價機iPhone 5C將登場。新iPhone外觀或與iPhone 5差不多，重大升級集中於處理器速度、加入指紋掃描、以及把系統提升至iOS 7。
