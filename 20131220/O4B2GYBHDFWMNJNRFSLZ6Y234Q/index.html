對沖人生：基金最後衝刺食尾糊 - 錢志健

兩個半交易日後便是聖誕，現在讓時光倒流至1999年聖誕、等待千禧年來臨之際。科網股大熱年代，上一代最「醒目」的操盤人，眼見某些公司的價值已超乎現實，因而拋空股票，卻弄到遍體鱗傷。

賺管理費非易事

三個月前筆者提及一位叫Henry Blodget的奇人，他曾是美林的互聯網分析師，在1999年聖誕，他說網上巨人Amazon會在一年內大升100美元。當時我有一位北美長短倉行家Ahmet Okumus不信邪，當Amazon做220美元時，他在250美元沽出Amazon的認購期權，想着有30美元的buffer，這sell call策略可能他大賺一筆。說時遲，那時快，當大事瘋癲時，上升動力可以無止境；Boxing Day後復市，Amazon狂升100美元，結果Okumus勁輸，一個交易日內資金曲線跌了29%，真慘情！
回到今天，當有人問及美股會否在明年深度調整等，我可以告訴你，一些對沖基金在年尾已收爐——現金為王、鎖定利潤；明年1月2日再拼過你死我活。普通人和機構投資者有一分別，就是絕對回報基金要盡量在年尾拿獎勵費（Incentive Fee）；「有錢分」是行業的「麵包與牛油」，並不是賺管理費那麼簡單。
現在轉談另一個話題。神級對沖基金經理也是以股票為膽，即core portfolio。根據Insider Monkey資料顯示，量子基金創辦人兼對沖大鱷索羅斯個人十大持股倉位，排行第一是標普500的ETF，第二是微軟，兩者也是派息股。老索在2000年退休後，實際再沒有管錢，他也是順勢而行，因他相信當世界仍然存在，金融市場也存在。坦白說，一些無心戀戰的對沖基金操盤人，個人投資漸趨保守，不願給別人管理費。齋買ETF，有時唔多work。另一神人「沽神」保爾森，黃金ETF及黃金類別股票在過去三年便令他輸了十億美元。其實操盤人多數有立場，並非完全中立。
我依然深信二級市場可製造財富。能在市場賺錢的操作者，並不會四處張揚自吹自擂，這句話用英文表達會更貼切：「Extract return little by little」，而並非做高槓桿的個別「大搏殺」，像上世紀九十年代初索羅斯集中火力沽英鎊般的成名之作。在此一提芝加哥CBOE VIX波幅期指「用量」再沒有屢創新高（落筆時在16點徘徊），是否基金經理已經不再hedge，認為對沖沒有用，還是已經收爐變現拒絕再玩？嘗試用你的想像空間吧！

破產才能發達？

在心戰領域，有炒家曾說，要有近乎破產的經歷才可浴火重生，所以他天天着迷買輪、長短期指、倫敦金等。有位世伯，七十有中，十年來炒金累積輸了100萬元，因為要有所謂的精神寄託，在這領域追逐日光，真是too bad so sad。我想在此一提《操盤手回憶錄》Jesse Livermore曾說：「市場只有一個方向，不是作多、不是作空，而是對的方向」。Well，這位上世紀初的一代股神經歷數次破產，最後更吞槍自殺，這又是個好的破產經驗嗎？
最後讓我用心靈話語作總結。我是個「不稱職」的上帝門徒，賠錢時有時也放不開；但想深一層，富貴如浮雲，市場的事，讓市場去解決便可以了。本年最令我上心的事，是香港未來33年是怎樣？香港是否有真普選，很在乎各位的參與。我不想把「佔中」與宗教混來一談，因我知道上帝會作最後審判。明年1月5日星期日下午，一個基督徒團體邀請我從中環人的角度分享《超越中環價值的價值重尋》，來一次信仰、政改及中環價值的對話，有興趣的讀者可聯絡我。
每年的平安夜（今年是下周二），我會約一些對生命救贖「半信半疑」的朋友到教堂，去聽聽基督降臨世上的訊息。我信上帝，我也相信人生是短暫的事情，尋找信仰的心靈雞湯，或會令我們更了解到生命本身就需要有「盼望」，從而找到平安。如果你不知道如何聽到盼望的信息，歡迎你來找我。祝各位聖誕快樂！
