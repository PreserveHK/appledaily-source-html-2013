椰子油vs腦退化 - 嚴浩

一位讀者問：「我爸爸患了老人痴呆接近兩年，一路惡化下去，大部分時間都不愛說話，醫生說他是前葉出現問題。」
幾年前有一位美國的兒科專家Mary Newport根據她丈夫的老人痴呆案例出了一本書，書名叫"What if there is a cure"，這案例又被美國CBN TV報道，引起西方世界從新評估腦退化的新治療法。Dr. Mary Newport在丈夫的腦退化無藥可治的時候，發現了椰子油有可能對腦神經之間的溝通有正面作用，這本書記錄了這個治療過程。更多的詳情包括服用椰子油的方法已經收錄在《嚴選偏方》第2集，有需要的讀者在互聯網上輸入Mary Newport可以看到她的blog。《半畝田》介紹椰子油治療腦退化的一年多來已經陸續收到正面的實戰反饋，建議這位讀者在沒有辦法之下也讓家人試試，底線是對健康沒有一點壞處。
椰子油有多種用途，以我自己的老媽媽為例，老人家已經成為半個植物人，這樣的病人一般都有便秘和抽筋的問題，我請家人把椰子油混進食物通過管道輸入胃中，每天三湯匙，一兩天就舒緩了便秘問題，抽筋問題需要長一點時間，當家人發現媽媽不抽筋了，也已經過了幾個月。在實戰中，我發現腦退化的病人有兩種狀況，一種比較安靜，一種比較鬧，似乎安靜類型適合用椰子油，但比較鬧的類型適合用冷榨亞麻籽油。
