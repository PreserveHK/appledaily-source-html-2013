避險升溫　美股早段反覆

【本報綜合報道】英國1月工業生產意外下跌，同時德國央行行長魏德曼表示，德國經濟仍受歐債危機困擾，對德國前景構成龐大風險。投資者憂慮全球經濟復蘇是否遇阻滯，避險情緒回升。美股早段表現窄幅上落，現貨金價則表現向好曾升近1%，試圖重上每盎斯1600美元水平。
英國1月工業生產下跌1.2%，遜市場預期升0.1%。英國1月製造業亦意外錄得收縮1.5%。消息拖累英鎊走弱，昨日曾跌0.56%至1.4831美元，創2010年6月以來最低。此外，德國央行行長魏德曼表示歐債危機的衝擊尚未完成平伏，投資者對歐元區的信心亦只回復部份信心，所以對德國仍構成很大風險。歐股尾段升幅收窄，英、法、德股分別升0.08%至0.33%。美股開市後45分鐘（香港昨晚10時15分）道指報14477點，升30點，納指報3248點，跌4點，標普500指數報1555點，跌不足1點。

格羅斯續減持美債

美股上周連續四日創新高，標普500指數亦撲向07年歷史高位之際，有「金磚四國之父」之稱的高盛資產管理部主席奧尼爾（Jim O'Neill）預測，短期內美國經濟數據表現佳，將有助美國股市上揚，但難以預測美股升勢是否能持續。
儘管美股後市方向未明，但「債王」格羅斯（Bill Gross）的投資旗艦PIMCO公佈，該公司的總回報基金（Total Return Fund）2月減持30%美國長債，持有國債佔總資產跌至28%，顯示他亦將資金陸續抽離債市。美國10年期國債上周五曾攀升至2.08厘的11個月高位，昨日曾升2點子至2.04厘。
