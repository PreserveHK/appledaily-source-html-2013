Galaxy Note 3　新機開盒率先睇

三星新機Galaxy Note 3，下周三才在香港正式發售，但有外國網站Phonearena已率先將Galaxy Note 3開盒片段上載。新機屬Note 3的32GB版，各位三星粉絲，一齊去片！
