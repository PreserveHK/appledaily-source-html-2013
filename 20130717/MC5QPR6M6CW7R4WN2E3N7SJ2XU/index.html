基金上月蝕逾2.6%
爆發贖回潮　債王求客留低

【本報綜合報道】聯儲局主席伯南克今年5月首次放風講退市後，觸發美國國債大冧價，正當很多基金經理眼見勢色不對，大舉將資金撤出環球債市之際，全球最大債市基金Pimco卻繼續溝貨，結果拖累旗艦基金上月錄得金融海嘯以來最大虧損，客戶贖回金額亦破紀錄。

據Pimco網站資料顯示，旗下Total Return Fund的投資組合，持有的6月美債持倉比重，由5月的37%增至38%，按揭證券持倉比重則增加2個百分點至36%，是2月以來最多。

格羅斯死撐債市已超賣

不過，由於伯南克上月議息後，出乎意料披露退市路線圖，揚言明年中或結束買債，環球債市又再捱沽，美國10年期國債由5月尾的2.19厘，升至6月28日的2.53厘，令PIMCO的美債持倉再蒙受損失。
根據Morningstar資料，Total Return Fund於上月錄得2.64%虧損，是2008年9月以來最嚴重，令今年迄今的損失擴大至3.11%，跑輸76%的同業。眼見基金表現欠佳，加上儲局退市陰霾下，投資者看淡債市前景，PIMCO客戶上月的贖回額達到破紀錄的96億美元，令基金管理的資產規模縮至2,680億美元。
雖然基金今年表現失色，但掌舵人、債券大王格羅斯卻不肯認低威，指基金40年以來的表現出眾，客戶不應為一兩個月表現欠佳而耿耿於懷，呼籲客戶繼續留低。他又提到，債市的低息時代可能已於4月份出現轉角市，但近期債市已呈現超賣，估計美國10年期國債孳息可回落至2.2厘。他周一於其網誌派定心丸，相信聯儲局在2016年才開始加息，揚言低息環境將會是債價的最大支持。

伯老出席聽證會成焦點

Total Return Fund於1987年5月成立以來，至上周五為止，每年平均回報率達8.04%，較反映大市表現的巴克萊美國整體債券指數的6.94%回報為高。
另方面，市場觀望伯南克今日出席眾議院金融服務委員會的聽證會作證時，會否就何時縮減買債發表新的言論，美國國債價格昨日窄幅上落，10期國債孳息在2.51厘徘徊。英國《金融時報》估計，伯南克可能面對國會議員更尖銳的提問，例如儲局何時開始縮減買債，美國經濟現況、債息上升會否影響儲局的經濟預測等。值得注意的是，由於聯儲局本月底的議息會議後，伯南克不會開記者會，換言之，今明兩日的聽證會，將會是9月議息前，伯老與外界溝通的最好時機，故市場對今日的聽證會格外關注。
