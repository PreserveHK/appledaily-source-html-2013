咖喱牛腩：
球品好‧人品好 - 牽牛

農曆新年假輕輕鬆鬆咁度過，又要投入工作；一如近年，香港都有兩大集團拍攝賀歲電影，不過同樣唔係我嗰杯茶，結果我只睇咗《虎膽龍威：擇日開戰》同《西遊．降魔篇》。其實港產賀歲片都唔係冇好嘢，N年前講打麻雀嘅賀歲片就好多金句，都𠵱家仍然係經典，特別係「牌品唔好，人品就唔好」。
放喺足球場呢？曼城中場耶耶托尼都算球品又好，人品亦好；周日英超大戰，耶耶同車路士嘅拉美利斯，就經常喺中場拼搶，兩人互相犯規鏡頭就少不免。其中一次耶耶明明精脆咁截斷拉美利斯，呢位巴西國腳當然加啲演技落去，博到球證吹罰，不過耶耶都冇大發脾氣，只係用拜神手勢向球證解釋。但轉到冇幾耐，耶耶一次推進，過埋拉美利斯，拉美利斯就起腳踢向耶耶膝頭，當事人梗係傷，球證亦都畀黃牌拉美利斯，不過拉美利斯竟然好唔憤氣，指住球證「F***」！
其實耶耶托尼喺曼城嘅作用，唔單止今仗入波咁簡單，佢直頭就係球隊中場嘅定海神針；貴為英超最高薪球員，不過曼城都準備向佢提供多5年合約，就知佢重要性，特別曼城1月聯賽狂失分，畀曼聯拋離，多少與耶耶去咗踢非國盃有關。
講耶耶嘅人品好，亦都反映佢嘅專業態度，非洲球員獲得大會高薪合約後，往往「hea態出現」，最經典就係艾迪巴約，熱刺一買斷佢，佢今季踢咗幾多場好波？不過耶耶就唔會，每場比賽都畀足120%精力，以後就算曼城換幾多任領隊都好，相信耶耶都係球隊至愛，亦都係佢下任隊長理想人選。
文：牽牛
