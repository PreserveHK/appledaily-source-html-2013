Galaxy Note 8.0板腦
迎擊iPad mini

世界流動通訊大會（Mobile World Congress）昨天（周一）起一連四日在西班牙巴塞隆拿舉行，各大電子產品商，都展出最新手機和平板電腦，挑戰蘋果地位。

南韓三星在MWC的主打新產品，是Galaxy Note 8.0平板電腦，主打可打電話和觸控筆，擺明針對蘋果的iPad mini。
採用8吋觸屏的Note 8.0，加入了電話通話功能，簡直是放大了的Galaxy Note 2。Note 8.0售價未定，機身厚7.95毫米，厚過iPad mini的7.2毫米，但1.6GHz四核處理器，規格比iPad mini的1GHz雙核高。
Note 8.0配備的S Pen觸控筆，令在板腦寫字和繪圖操作更精細。三星也加入不少新核心功能，像同時查看多個執行中應用程式的「多重視窗」，及自動調整最佳屏幕解像度的「閱讀模式」。
