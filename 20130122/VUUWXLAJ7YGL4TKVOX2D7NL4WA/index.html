真的要為港女來個大平反！係！可能我們真的比較喜愛Shopping，但並不代表我們一律都是虛榮、對廚藝一竅不通。今次Blaire、Claudia和Jasmine齊齊向Haze學整港式小食的變奏版──焦糖曲奇蛋撻。
港式蛋撻分為酥皮和曲奇皮兩種，不過酥皮比較油膩，而且買塊急凍酥皮就得，毫無挑戰性，三位靚女話誓要挑戰曲奇蛋撻。而且市面上的蛋撻出爐時間比較早，真是遲了一點起床就吃不到新鮮出爐的蛋撻。與其晨早起身出街買，倒不如做做淑女，閒時在家玩玩入廚樂，向大家示威兼平反之餘，更可一嚐新鮮蛋撻的滋味。今次更會將Creme Brulee（焦糖燉蛋）焦糖脆面的概念加入蛋撻中，令蛋撻變得不平凡。別以為很簡單，火槍位置要對得準才不會燒焦曲奇皮。Haze更會以三位製作蛋撻的外觀、味道和技術加以評分，全程絕不加以援手，以見三位的真功夫！
除此之外，Haze還會示範多兩款經典港式小食的變奏版，教大家如何做出帶有回憶得來，又有點驚喜變化的Hong Kong Style小食。

焦糖曲奇蛋撻

講起港式小食，又怎能不提蛋撻呢？香酥的牛油蛋撻皮加埋幼滑的蛋，面層更加上脆脆焦糖，將法國的Creme Brulee融入港式蛋撻的美味中，簡直是雙重享受！

份量：8個　時間：45分鐘　難度：★★★★

材料：
曲奇皮材料：

糖霜40克、牛油100毫升、蛋白25克、低筋麵粉150克、奶粉15克、呍呢拿油少許

蛋漿材料：

牛奶80毫升、水280毫升、蛋4隻、糖80克、呍呢拿油少許

焦糖面材料：

糖適量

做法：
蛋漿做法：

1. 燒滾水煮溶糖，待涼後加入牛奶及蛋拌勻。

2. 用篩過濾蛋漿的雜質及蛋泡，滴入呍呢拿油拌勻備用。

曲奇皮做法：

1. 先將牛油和糖霜拂勻，拂勻後加入蛋白、奶粉和呍呢拿油拂勻，再加入已過篩麵粉搓勻。

2. 麵糰用棍碾平，用圓模裁出適用大小鋪在撻模上，用叉在撻皮中篤上細孔並將蛋漿倒入撻皮中。

3. 將（2）放入已預熱180℃的焗爐，焗約15分鐘取出。

4. 最後於撻面灑上糖，再以火槍將撻面燒成焦糖即可。

有餡菠蘿包

還記得電影中的經典對白：「你估菠蘿包入面真係有菠蘿餡咩！」對啊！誰說菠蘿包不可以加入菠蘿餡呢？除了外形十足菠蘿，餡料更加入啖啖菠蘿，食落更鮮甜！更何況自己做的菠蘿包，一定比市面上連鎖麵包店賣的「空氣包」更實在！

份量：8個　時間：3小時　難度：★★★★★

材料：
菠蘿皮材料：

糖80克、食用臭粉1克、食用梳打粉1克、低筋麵粉120克、奶粉10克、牛油60克、蛋25克、淡奶10毫升、呍呢拿油1克

餡料材料：

吉士粉30克、奶150毫升、菠蘿肉（切粒）適量

菠蘿包材料：

高筋麵粉225克、糖40克、奶粉6克、依士4克、水110毫升、蛋30克、牛油20克、鹽1克

做法：
菠蘿皮做法：

1. 將粉類混合過篩備用，另將牛油和糖拂勻後，拌入粉類攪拌，再加入蛋、淡奶和呍呢拿油拌勻，不可用力搓，以免起筋。

2. 混合後放入雪櫃冷藏1小時。

3. 將菠蘿皮切成小塊，搓圓後壓平，放上麵糰上。

餡料做法：

將吉士粉加入奶拌至沒有粉粒，加入菠蘿肉拌勻即可。

菠蘿包做法：

1. 先將所有材料攪勻及搓至滑身。

2. 進行第一次發酵約60分鐘，至麵糰2至3倍大。

3. 按壓麵糰進行排氣，再分割成8個小麵糰。

4. 搓圓小麵糰，再靜止麵糰10分鐘。

5. 再將麵糰造型後包入餡料並進行第二次發酵約45分鐘。

6. 放上菠蘿皮，掃上蛋液，用牙籤於菠蘿皮上𠝹出井字狀後放入預熱190℃焗爐焗15分鐘，完成。

小貼士

菠蘿皮應早一天製作，讓臭粉有足夠時間令菠蘿皮產生發酵作用，這樣焗出來的菠蘿皮紋更好看。
要檢查麵糰是否發酵好，可將手指沾上麵粉插進麵糰中，若拍孔沒有收縮，表示已發酵完全。

三色缽仔糕

小時候放學，總嚷着外婆給我買個缽仔糕，還會指定吃「啡色，少豆」，即是黃糖缽仔糕，少紅豆。不過可能細個吃得太多，現在看見缽仔糕都有點滯，一於學做變奏版的缽仔糕，加入清醇的綠茶和香甜的士多啤梨口味，令砵仔糕變得更清新可口。

份量：9個　時間：45分鐘　（未計處理紅豆時間）　難度：★★★

材料：

粘米粉80克、澄麵70克、砂糖200克、清水650毫升、紅豆/綠茶粉/士多啤梨粉各適量

做法：

1. 紅豆先用水浸2小時，之後以400毫升滾水煮約1小時至腍身，隔水備用，煮豆水留用。

2. 粘米粉及澄麵過篩兩次後加入250毫升清水拌勻成粉漿，並分成3份，其中2份分別加入綠茶粉及士多啤梨粉拌勻。

3. 將400毫升煮豆水加入糖煮溶，隨即把粉漿沖入糖水中，立即關火並不斷攪拌成稀糊狀。

4. 最後將麵糊倒入碗內並加入紅豆，以大火蒸約20分鐘，完成。

小貼士

把粉漿沖入糖水的步驟一定要快，否則缽仔糕就不夠彈牙。
