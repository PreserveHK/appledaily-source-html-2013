美麗孕媽：懷孕可以吃甚麼？

最近收到讀者報喜說陀了BB十幾周，真替她興奮又高興！雖然生小孩的那刻很痛，湊baby的頭兩個月很艱苦，但當日子一天一天的過，看見寶貝囡囡的樣子越來越精靈，笑容越來越多，多想立即生多幾個小寶貝一起玩！（老公看到這裏一定在冒汗）。這樣說來，旁人可能以為生小孩很簡單，而且猜想懷孕時可以大吃大喝，多爽！可惜這一切都只是誤解，因為陀B最煩惱的，正是每天在研究自己可以吃些甚麼。懷孕時期，健康均衡的飲食更見重要，而且很多食材其實都「不准碰」，例如：生冷食品、含咖啡因的飲料、含水銀的魚類、醃製食品、寒涼食材等等。
準媽媽如我的讀者一樣，都正在煩惱懷孕時的餐單，所以今期讓我介紹幾款菜單給各位靚媽咪，這些都是我懷孕時常吃的！
早餐：牛油果番茄熟蛋三文治 / 高鈣低脂牛奶煮燕麥片配雜莓 / 南瓜雞肉奄列
午餐或晚餐小菜：上湯腐竹浸魚柳 / 蓮藕片炒牛肉 / 菜粒肉碎燴豆腐 / 花菇鮮菌煮雞柳
湯水：蘋果雪梨煲豬骨 / 淮山百合栗子瘦肉湯 / 鮮奶花膠燉雞湯
飲品：杞子雪耳雪梨茶 / 百合蓮子南棗糖水 / 蘋果薑茶
