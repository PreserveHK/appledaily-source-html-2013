眼睛想旅行：屍奔台北　拔足喪追人

臨近萬聖節，血淋淋喪屍將聚集台北，因為首屆「殭屍路跑」Zombie Run即將舉行。源自喪屍電影情節的「屍奔」活動在美國大城市都有定期舉行，在賽跑中加插喪屍追趕攞你命情節，可激發體內的潛能之外，同時也為有需要人士籌款。今次台北版亦一樣，參加者可選擇做人類被喪屍追逼，或者直接做殭屍索命趕人；人類跑手腰間會綁上三條代表性命的紅絲帶，在5公里路程中，只要保得住命仔跑完就算成功，否則就被感染成喪屍！活動預計有三千人報名，一齊在台北街頭Fun Run衝衝衝！

記者：劉鎮豪

Zombie Run Taipei 2013

地點：大佳河濱公園
網址： http://www.zombierun.com.tw
