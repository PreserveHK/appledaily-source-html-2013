Samsung平板夠堅　縮水iPad迎戰

於德國柏林消費電子展上，Samsung不單發佈新手機及新手錶，亦推出新一代Galaxy Note 10.1平板電腦，雖然風頭給Galaxy Note 3搶去，但此平板仍值得留意。
記者：梁海威

新Galaxy Note 10.1，同樣採用Galaxy Note 3的仿皮外殼，手感極佳。而Note3擁有的功能，如S-Pen、Air Command、Scrapbook、多重視窗等，新Note 10.1都全部齊備，可謂Note3的放大版本。硬件方面，4GLTE版會採用2.3GHz Snapdragon 4核處理器，3GB記憶體，配合2560 x 1440解像度的LCD屏幕，絕對是高級平板的規格。新機預計今年第3季發售，價錢尚未公佈。

此外，有網站發放短片，表示收到一部疑似iPad第5代的機殼。從片段中看到，疑似機殼比上一代iPad更薄，體積更小，屏幕仍然保持9.7吋，但邊沿位置縮窄，所以整個面板亦縮小。且看蘋果公司行政總裁Tim Cook，會否於下周發佈會展示新iPad。
