容許再生：跟着韓劇畫出無公害妝

最近，亞洲掀起韓風潮，從去年整年爆紅的「江南Style」，讓許多外國人在網路上也跟着大跳江南大叔的流行舞步，加上多部穿越戲碼的韓劇火熱飛來，韓國熱似乎成了全民運動。但看了這些韓劇，老牛其實大部份並不是在看這些，我最近在研究韓國女明星在韓劇中畫的妝，從10年前到現在其實改變很大，尤其是這一兩年的韓劇，仔細看你會發現，韓劇女主角的妝都是走一種看不出有化妝的「無公害妝」，這種妝容的厲害之處在於，其妝容的質感簡直媲美天生麗質的偽素顏妝，讓你在看韓劇時，不但為劇情傾倒，更為她那美麗的皮膚膜拜！到底這樣的無公害妝要掌握哪幾個條件呢？讓老牛來一一述說。

Key Point 1：無公害的超自然緊緻底妝

一抹以往韓劇金喜善年代喜歡厚重底妝的質感，看看現在這些韓劇女主角如尹恩惠、宋慧喬等等在去年與今年的兩部爆紅韓劇裏頭，無瑕輕透的底妝簡直擄獲了所有熱愛韓劇影迷的心，說穿了，就是完全超輕薄的無瑕底妝，沒有任何粗大的毛孔、痘痘或是油光滿面、擁有面部不浮腫的超緊緻膚質，如何創造呢？其實韓劇女星們，本身的膚況就保養得很好外，仔細端倪你會發現，她們的肌膚都很水嫩飽滿，想必是在事前的保養工作做得很好，才能讓後續的底妝如此完美。所以，想畫一個完美的無公害底妝，妝前保養就是第一步。首先，洗完面後，可以先用精華液或精華乳霜調和一點美容油，做一個三分鐘的面部按摩，可以活絡肌膚循環並達成V面的效果。

老牛的V面按摩，加速緊緻水嫩面養成！

No.1　揉散脂肪聚集：
以拇指與食指的「雙V」指腹，從眼下至下巴輪廓，以內往外橫向揉捏，以揉散脂肪堆積。活絡面部血液迴圈，視為塑面前的熱身操，可以來回兩次。

No.2　收緊眼尾紋：
以拇指與食指的「雙V」指腹，拇指抵在太陽穴，食指從眼頭往眼尾輕柔的滑動上提，做到緊提眼尾紋的作用，雙眼可來回五次。動作要特別輕柔，以免動作過大反致細紋產生。

No.3　撫平法令紋：
以拇指與食指的「雙V」指腹，拇指抵在太陽穴，食指從鼻翼兩側往拇指方向斜上滑動，可收緊頰肉，達到撫平法令紋的作用，可來回五次。

No.4　緊塑縮下巴：
以拇指與食指的「雙V」指腹，拇指抵在耳前凹陷處的淋巴結，食指從下巴往拇指方向斜上提拉至嘴角兩側，可緊塑下巴鬆弛、幫助排水、消水腫，達到微塑V面作用，可來回五次。

No.5　微塑V字面：
雙手手掌張開但手肘貼合像支撐下巴般，從下巴根部往上移動提拉，可緊塑鬆弛輪廓、完美打造V形面，可來回五次。

Key Point 2：無公害的根根分明睫毛

當年日本的超濃密睫毛、激黑眼線，現在完全不流行也不適用啦！仔細端詳韓劇女主角裏面的睫毛，絕對是根根分明，而不是濃密到看不到眼睛的清澈度，所以女主角總是能夠展現無辜的自然大眼，眼睛其實都經過了精雕細琢卻看不出其誇張度，也是整體妝容能看起來超級無害的原因之一，但可能很多女生會問，我就是沒有濃密的大眼那怎麼辦？其實是有技巧的，除了先把睫毛夾翹外，要改變以往用整排假睫毛的方式，建議你改成一株一株種睫毛，把睫毛間的空隙統統慢慢有耐心地種滿，之後再用濃密纖長型的睫毛膏將真假睫毛一起輕輕刷上，就能展現展現又翹又輕薄的睫毛。

Key Point 3：無公害的俏美唇

豐唇在韓國一直都是被大家公認的微整形重點，今年的無公害俏美唇，首選的唇色是裸偏淡珊瑚粉色系的美唇，最佳參考就是宋慧喬主演才剛推出的電視劇《那年冬天，起風了》裏頭的喬妹，唇色無敵自然，但重點是化妝師掌握了她唇部的水嫩度和顏色的淡雅度，平常我們該怎麼辦到呢？首先，如果你是唇色淺的，直接找一支淡雅裸色的粉嫩唇膏即可，如果你是唇色深的也不要緊，為了讓你的唇膏色澤能顯現在唇上，就要用遮瑕膏或一點點粉底先蓋掉你原本的唇色，然後選擇水潤度保濕度高的唇膏，將上下唇均勻塗滿。最後，別忘了用同色系的唇蜜多塗抹上一層，就能創造裸珊瑚粉色的自然唇色。
