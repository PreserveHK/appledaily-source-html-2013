超越Galaxy Note 3傳iPhone 研究6吋大芒

超越Galaxy Note 3
傳iPhone 研究6吋大芒

面對三星手機的挑戰下，蘋果公司早已向大屏幕策略屈服，消息人士指，蘋果正研究推出更大屏幕的iPhone，考慮中的一款屏幕最大達6吋，超越三星Galaxy Note 3的5.7吋芒。
iPhone堅持了五年的3.5吋芒在去年推出iPhone 5時已經打破，變成4吋芒，消息人士指，蘋果將進一步研究推出不同尺寸屏幕的產品，零件供應商指蘋果近月已開始測試4.8吋至6吋的屏幕，目前蘋果對4.8吋屏幕尤感興趣。

iPhone 5C代工廠涉虐勞工

蘋果下周二將推出平價版iPhone 5C搶佔市場，但推售前傳出負面消息。基地在紐約的勞工權益組織「中國勞工觀察」昨日指出，美資捷普集團在中國無錫生產iPhone 5C背面膠殼的代工廠涉嫌虐待勞工，工人每月被迫加班逾100個小時、每天需站立工作逾11小時，只有一次30分鐘的休息時間等。蘋果表示，已派員調查。
德新社/美國《華爾街日報》
