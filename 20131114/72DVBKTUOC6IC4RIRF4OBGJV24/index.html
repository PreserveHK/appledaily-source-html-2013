蘋論：國安會將成港人夢魘 - 盧峯

1947年7月26日，美國總統杜魯門簽署《國家安全法》（National Security Act），成立了國家安全委員會（National Security Council）及一系列國安、情報機關、中央情報局（CIA）也依據有關法例成立。當時正值二戰後蘇聯勢力不斷擴張，世界局勢變動不定，局部戰爭以至全面戰爭隨時來臨的危險時刻。
2013年11月12日，中共第十八屆三中全會宣告成立「國家安全委員會」（國安會），由中共總書記兼國家主席習近平親自掛帥，成員包括公安、國安、外交部以至港澳辦負責人。跟1947年的動盪局勢相比，此時此刻國際形勢雖不能算風平浪靜，但大國間發生全面戰爭的風險甚低，中國受到外來的直接威脅不大，倒是中國的近鄰對她的迅速崛起及頻繁軍事活動有點惴惴不安。那為甚麼中共最高層要在這個時候特意成立國安會，把國家安全議題提升為今次三中全會的主調呢？
最根本、最重要的一條就是加強控制。加強控制主要有兩個層面：黨內及黨外。首先說黨內。習近平成為第五代領導核心絕不是風平浪靜的，前重慶市委書記、政治局委員薄熙來及他的政治派系就對安排有異議，還鼓動反對力量準備大幹一番。最後薄熙來雖失敗而被打倒及判刑，但他的強大政治能量反映中央政府包括最高層對懷有異心的地方諸侯無力控制，甚至所知不多。試想想堂堂世界第二強國居然發生高官叛逃到美國領事館事件，還得美方合作才順利解決。北京最高層怎能不感到丟臉、不感到顏面掃地呢？
成立類似前蘇聯KGB的國家安全委員會形同在中央領導核心特別是總書記下設立一支情報親兵，直接受他指揮，直接到各處蒐集情報，足以穿透地方、部委黨組織。事實上KGB當年厲害之處正在於它不受其他黨組織限制，可以任意行事及監察黨高層，收集地方諸侯把柄，令他們不敢再輕舉妄動。
除了加強監視黨內不同派系外，成立國安會當然也志在加強對社會的控制，當然也會監察社會上的異動，以把反對力量消滅在萌芽階段。前不久在北京天安門廣場發生的自殺式襲擊雖未造成重大破壞及傷亡，但已清楚顯示內地少數民族爭取獨立自主的聲音並沒有因強力鎮壓而退卻，反而有越來越激烈的趨勢。再加上互聯網、流動通訊普及，監視、箝制異議者比以前更困難及費力，中央最高層自然要提升處理國安問題的層次，好增加可用的資源，好把監視、控制之手伸的更長更遠。
最令港人擔心的是國安會把主理港澳及台灣事務官員納入架構內，變相令國安會職權涵蓋港澳事務。要知道國家安全很容易成為收緊箝制公民自由及權利的藉口，03年50萬人上街反對就《基本法》二十三條立法，就是要防止特區政府制訂惡法侵損市民的言論、集會、新聞自由。現在，中共最高層成立國安會，名正言順把港澳台列入職權範圍，意味國安會屬下的部門及人員將在港澳地區大展拳腳，蒐集情報，並可能特別針對一些向來被視為眼中釘的組織、政黨、議員及學者進行監聽，又或是對某些政治行動如和平佔中、7.1大遊行搞分化破壞。這不是令香港的自由空間，令港人的公民權利蒙上極大的陰影嗎？
中情局前僱員斯諾登及較早前維基解密揭露的資料顯示，即使在民主及講法治的國家，國家安全部門及情報機關仍不時出現違法監聽及截取通訊的行為，甚至友好國家總理的手提電話也不能倖免。中共是個控制壓倒一切的專權政體，她成立的國家安全委員會只會更肆無忌憚損害公民自由，只有更無孔不入的滲透社會各階層，只有更可怕！
