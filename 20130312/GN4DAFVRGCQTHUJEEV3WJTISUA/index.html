盡現甘甜

加拿大水質十分優良，而且漁民於大海放養，出產的海鮮有海水甘鹹，又有海鮮原來的甜味。今日就請來常駐當地的創意大廚Denice Wai教大家將其龍蝦、魚類等海產，以特別的中菜方式烹煮，讓海產的甘甜得以展現！
記者：張文蕙　攝影：林栢鈞　加拿大海鮮售賣點：city'super （2736 3866）

加拿大環繞北冰洋、大西洋和太平洋之間，境內有五大湖，擁有世界最長的海岸線，所以也擁有世界上很有價值的海產業，而且海水未被污染，令海產都特別鮮甜。行本地街市，雖然檔檔都話有加拿大海鮮，不過不一定正宗。近日加拿大駐香港總領事館聯同加拿大農業及農業食品部，推出iPhone應用程式，令本港消費者可輕鬆地搜尋加拿大美食。只要在Apple App Store搜尋CanadaFood.hk，便可免費下載此應用程式。

龍蝦甜糕

加拿大自1870年起就開始龍蝦的出口產業。龍蝦為加東海岸的特產，蝦農以6公尺見方的養蝦籠置於海中養殖，所以龍蝦特別鮮甜，亦有海水的淡淡甘鹹味，味道多變。Denice將它化成甜品，以玫瑰花片弄成如像宮廷糕點，龍蝦的鮮甜更突出了！

時間：1小時30分鐘　難度：★★★★　份量：3人

材料：
加拿大龍蝦半隻、乾玫瑰花1茶匙、冰糖90克、魚膠粉1.5湯匙、水300毫升

做法：
1. 龍蝦放尿後原隻灼熟，浸於冷水一會。

2. 拆去頭腳，揭開背部的殼後取肉，切片備用。

3. 水煮滾後將火調慢，加入冰糖煮至完全溶解。

4. 關火放入魚膠粉拌勻，將已弄碎的乾玫瑰花加入。

5. 放涼後，將半份混合液倒入啫喱模中。

6. 雪至半凝結鋪上龍蝦肉，再倒入餘下的混合液，雪至凝結即成。

醉青口

青口生長在加拿大大西洋省分，乾淨及冰冷的大西洋北部和太平洋的海洋提供了理想的棲息地。青口含豐富的蛋白質和礦物質，低脂肪，以花雕酒浸煮後，酒味仍不會覆蓋青口的鮮甜，甚具中式風味！

時間：5小時20分鐘　難度：★★　份量：4人

材料：
加拿大青口1,000克、花雕酒250毫升、香茅1株、青檸葉3片、南薑2片、鹽/糖各適量、水700毫升

做法：
1. 青口洗淨，濾乾水份。

2. 將香茅略拍爛。

3. 香茅、南薑、青檸葉放入鍋中，倒入水煮滾後，轉細火煮10分鐘。

4. 放入青口煮至熟透（約3至5分鐘），以鹽、糖調味。

5. 關火後盛起，加入花雕酒，放涼後冷藏5小時即成。

黃蓋鰈魚柳賽螃蟹配黑醋汁

黃蓋鰈又名孫鰈魚，是於加拿大淺海很常見的魚類，經常出沒於北海約100米水深的海沙上。其貌酷似比目魚，卻由於肉質不及比目魚滑嫩，懂得烹調的人很少，以致捕魚者利用拖網捕捉鱈魚時，往往因為缺乏商業價值，把網內的黃蓋鰈丟回大海中去，但這樣牠們還是大量死亡，浪費了。其實只要以黃糖、生抽及老抽醃10分鐘再煎熟已很好味，但切記時間過長會令肉質口感變差，配合滑滑的炒蛋白來吃，口感更覺豐富！

時間：25分鐘　難度：★★★　份量：2人

材料：
加拿大黃蓋鰈魚240克、生抽/意大利黑醋各100毫升、老抽50毫升、黃糖200克、蛋白2隻、雞湯3湯匙、鹽/栗粉/黑醋/油各適量、黃糖60克、粟粉10克

做法：
1. 加拿大黃蓋鰈魚開邊去骨取出魚肉片。

2. 以生抽、老抽及100克黃糖醃魚柳10分鐘。

3. 將魚柳煎至甘香備用。

4. 於鑊中下油至起煙，將已拌勻的蛋白、鹽、雞湯快速炒熟備用。

5. 將黑醋、100克黃糖及粟粉拌勻後，以慢火煮至濃身。

6. 將（3）及（4）上碟，淋上（5）即成。

比目魚釀糯米飯

加拿大比目魚是紐芬蘭的傳統魚穫，肉質雪白的比目魚，鮮甜而脂肪含量富含三酸甘油酯，用它來做糯米飯，魚汁都浸入米粒內，加上荷葉的香味，美味得很呢！

時間：35分鐘（不計浸食材時間）　難度：★★　份量：4人

材料：
糯米100克、炸乾葱4粒、瑤柱20克、加拿大比目魚200克、糖2茶匙、乾荷葉2塊、生抽/老抽/芝麻/黑醋/水各適量

做法：
1. 糯米浸一晚後，於煲中放入糯米及水（水剛蓋住糯米），中火煮至全熟。

2. 炸乾葱切片，瑤柱以暖水浸軟。

3. 將（2）落鑊爆香後加入（1），以適量生抽及老抽拌勻待用。

4. 加拿大比目魚開邊去骨取出魚肉片。

5. 以4茶匙生抽及2茶匙老抽及糖將比目魚柳醃10分鐘。

6. 乾荷葉以暖水浸軟，印乾水份。將一半糯米飯鋪在上面，放上比目魚後再鋪上餘下的糯米飯，將荷葉包好。

7. 蒸7分鐘上碟，以芝麻及黑醋伴碟即成。
