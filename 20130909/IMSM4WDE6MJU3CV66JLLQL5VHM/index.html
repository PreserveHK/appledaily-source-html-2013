「禮儀之邦」 - 李純恩

「禮儀之邦」 - 李純恩

在大陸，每逢有些知名人士出席公眾場合活動，會有一些保安人員護在身邊開道，保安的態度都很惡劣，開道之時又推又撥，如狼似虎。
於是你看見很滑稽的場面，名人滿臉笑容對着大家走來，迎着擁躉或者客人們的歡呼，但他身邊卻是一群如狼似虎的保安，把那些迎上來的人推撥得又跌又爬，世界的美好和兇惡便同時展現，尷尬得令人難受。
但是，如果沒有這些如狼似虎的保安，那些歡呼迎上來的人卻又馬上如狼似虎起來。他們為了可以跟那名人握個手合個影，會毫無禮貌地把名人身邊的人包括他的家眷和朋友推撥得一跌一爬。香港一位大作家的太太跟我說，不知多少次了，她跟丈夫在大陸參加活動，有些女人擠過來，一側身就將她推開，然後站在大作家身邊「愉快合照」。更有甚者，將她擠開的女人還會把相機遞給她：來，你幫我拍一張！
於是在這個時候，即使你再討厭那些如狼似虎的保安，也會希望他們馬上現身，來對付這些毫無禮貌如狼似虎的擁躉。
這也是中國人禮節上的一個死結。因為不禮貌成了生活中的一種常態，於是禮貌好像也失去了意義。當大家都習慣了不禮貌，也就沒人對不禮貌見怪，反而不習慣禮貌。於是客人在餐廳有禮貌地叫伙計沖茶他不理你，你大喝一聲：沖茶！他拎着水壺就奔過來。你大喝只因想沖到茶，他被你一喝，只會覺得你真是等急了，也不會怪你沒禮貌，因為他沒有禮不禮貌的概念。於是保安不如狼似虎，客人就如狼似虎了。對於外來者，還真不能一口咬定誰對誰錯。
