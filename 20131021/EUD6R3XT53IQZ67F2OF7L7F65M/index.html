BIG SPENDER：老爺車開P

香港老爺車會自1979年成立至今，每年10月都會在中環遮打道舉辦一年一度的老爺車展。本月13日舉行的車展，共有91部老爺車參加，當日展出最古老的老爺車是嘉道理家族擁有的1926年賓利3 litre房車。參加者可以親身登上每部老爺車，吸引過千名老爺車迷入場，令本來已不算寬闊的文華東方酒店至昃臣道一段的遮打道行人通道，以及三條行車線擠得水洩不通。

車展中，評審會選出並頒發10個獎項予參賽的優勝車輛，和一個由所有參加者一同投票選出的最渴望得到的老爺車大獎。香港會計師公會前會長周光暉已連續四年參與活動，今年更首次憑一部1963年的勞斯萊斯Silver Cloud III獲最佳展出大獎。

記者：吳宛蔚
攝影：謝榮耀
部份照片由受訪者提供

今屆老爺車展，共有28部老爺車參賽，參展的亦有63部，擁有兩部老爺車的周光暉，今年已是第四年參加車展，不過還是首次贏得老爺車會頒發的獎項。
四年前他的Triumph TR6被評審批評過改用深藍色矽膠取代原來的黑色膠來封發動機，被扣分，評審由老爺車會委員會成員、車商、收藏家、車手及資深汽車專欄作家等組成。故今次他學聰明了，先請維修師傅塗黑膠帶，想不到今次卻被挑剔更換了顏色不符的火嘴線，「評審大部份都偏好原汁原味，我是求品質的人，只要有又新又好的代替品一定會換，不會為獎而妥協」。
周光暉視一年一度的老爺車展為年度盛事，他說老爺車會每年舉辦近30個活動，就只有車展和年頭的周年聚餐會將其排在公事及私事前，總會抽空出席。認真的他，在車展前一日就花了一整天時間，與兩位兒子一同到位於天水圍的車房作最後打點，抹淨車內外各機件上的污垢、掛上車頭徽章，為免展出當日早上出亂子，更在展覽前一晚，預早將車開到公司位於中環的停車場停泊。大會指引請他早上8時報到，他7時15分便到，「這裏沒有劃位，早到才爭取較佳的位置」。他成功把兩部老爺車：1973年的Triumph TR6，及1963年勞斯萊斯Silver Cloud III泊到最近入口位置，特別將整個修復過程的相片，整合成一張濃縮的圖片集，又帶來古董車品牌的年鑑及維修手冊來佈置。

周光暉花兩年維修愛車

周光暉首次參展的勞斯萊斯Silver Cloud III，是花了兩年時間才修理好的寶貝。1963年出產的勞斯萊斯Silver Cloud III，是車廠在1955年生產來取代Silver Dawn的Silver Cloud系列第三代，比前一代的Silver Cloud II，汽車外形作了輕微的調整，令車身輕了100公斤，發動機、化油器都得到改善，令整體的動力有7%的增長。
這輛勞斯萊斯由英國著名的車身製造公司James Young負責製造，全人手裝嵌的Silver Cloud III全球只有81部。這部車身用上英國皇室御用顏色（Royal colour），黑頂配紅番石榴色的長陣Silver Cloud III，是周光暉在2010年10月，趁留學英國的第五個兒子放假，特別跟他一同前往倫敦，參加以占士邦1964年的Aston Martin DB6為主展覽品的老爺車拍賣會時買的。
他所買的Silver Cloud III曾在英國的博物館擺放八年，很多機件早已壞掉，所以由機件、外殼至車內的木裝飾及地毯，他全部都得翻新及修理，只有真皮座椅保存得宜，簡單塗上皮膏就可。經常往返英國的他，小至一口螺絲釘，大至擋風玻璃，他都一概從英國入貨。他會趁沒有應酬的午飯時間，獨自駕車由中環的辦公室前往天水圍的車房，帶同飯盒與維修師傅研究及跟進維修情況。為了得到汽車的維修手冊，他加入了英國的勞斯萊斯車會Rolls-Royce Enthusiasts' Club，又從汽車雜誌中找到全球最大規模的古董勞斯萊斯零件供應商Flying Spares。
周光暉說Flying Spares大部份的員工都曾服務勞斯萊斯車廠，對不同年代的勞斯萊斯零件十分熟悉，亦與車廠關係良好，只要車廠生產的汽車有瑕疵，必定送到那邊劏，所以他們有較多原廠零件，而且會用原廠廢棄的模具，製造原廠委託製造（OEM）零件。

翻新Silver Cloud III全車木件

最考功夫的就要數安裝煞車系統，一般的煞車系統使用雙迴路煞車，一邊負責分前左後右，另一邊負責前右後左，但60年代的勞斯萊斯因多為國家元首及富豪乘坐，卻用上最安全的3組煞車技術，維修師傅都未處理過，故用了3個月時間，才摸索到安裝的辦法。
自迷上老爺車，每年11月他必定要到英國伯明翰參加一年一度、在National Exhibition Centre舉辦的老爺車博覽會，場內的參展商來自世界各地，展出各式各樣的老爺車零件、維修工具和汽車品牌的商品，如帽子、衣服等。去年，他正為維修車內的胡桃木傷腦筋，卻在展館裏找到一間公司願意把他的車內所有木件翻新，收費3萬元，不過要他負責運費。他想了想，最後還是擔心難以跟進而打消念頭。最後，他誤打誤撞地到了深圳車房林立的桃園路，找來一間專門做汽車翻新的公司，用數千元翻新50條木條。他堅持這次維修在低成本下完成，三年前，他以約65萬元買回來的勞斯萊斯，維修及翻新用了數十萬元，總投資逾百萬元。
至於他寶藍色1973年生產的Triumph TR6，則是當年車廠最受歡迎的車款，當時生產約94,000部的TR6，只有8,000多部分配到英國本土銷售，其餘均出口，在英國找到品質好的不容易。身為英格蘭及威爾斯特許會計師協會理事及商務委員會副主席，他每年最少往英國開會六次，當決定找尋年輕時開過的Triumph TR4A不果，他開始放眼去找代替品，最終在英國購車網站找到外形相近且性能好的TR6，拜會過車主知道他是車房東主，把車保養得宜，二話不說就決定買。有了兩次英國購老爺車的經驗，連好朋友都請他幫忙。
老爺車是花時間、花金錢的玩意，所以他有了退休的念頭才開始，「Swinging Sixties是我出生的年代，流行開跑車，Aston Martin、Jaguar 的E-type及Triumph的TR和MG系列、保時捷911等都是有錢人的玩意」。當年他沒有錢買，現在它們都成了老爺車，讓他年輕時的願望成真。「如你喜歡跑馬拉松又怎會介意出汗？」開老爺車亦然，縱然不如新款汽車般易控制，但能坐上一部心儀已久的愛車上又豈會有半點怨言？
