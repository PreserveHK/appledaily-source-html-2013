開波Kick off ：點解要跑？ - 余翠怡

最近看到一個運動品牌廣告，主題是跑步，自己也是該品牌的粉絲。整個廣告都是不同年齡的大人細路長者，更有視障人士，在晴天雨天白天黑夜不停跑步的背影，配上輕快的音樂，廣告末段出現一句slogan: 「Let the Run Tell You Why」。看完後真的有衝動立刻換上跑鞋出街跑步。
跑步，我相信是很多香港人喜歡參與的運動。一來，跑步是很個人化的運動；二來，跑步不受時間及天氣的限制；三來，跑步比較大眾化，不用配備很多專業器材，又不需特定場地，只要找一對舒適又價錢相宜的跑鞋，便可出發。很多人說跑步是會上癮的，2012年公佈的一項調查發現，跑步是20至59歲成人最常參與的運動，每年一度的渣打馬拉松也是跑者增加的主要原因。
很多人開始跑步是為減肥、為健康、為比賽、為陪朋友、為有型，為慈善籌款……甚麼理由開始也好，最後都會變成因為喜歡而跑，或者是為了一份堅持而跑。我媽媽兩年多前開始跑步，每天6點都去跑，偶爾一兩天因為天氣差沒去跑，她真的會感到失落，覺得自己沒有完成目標似的；有時她會抱怨很累，明天還是休息一天不跑吧，結果到了早上她還是去跑。她沒有想過參加比賽，因為她覺得自己跑得不快，但她喜歡跑步，好像重拾了年輕的感覺。
我看到媽媽愛上跑步後的快樂，或者，這就是她跑步的原因。
余翠怡
