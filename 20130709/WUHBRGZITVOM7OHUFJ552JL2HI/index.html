環島四圍轆之一：一千三百公里　踩逐山海夢

「做人如果無夢想，同條鹹魚有乜分別？」多得星爺鼓舞人心的一席話，不少人勇敢跳出生活框框尋夢去。證明自己非鹹魚類的舉動之一，叫做「單車環島」——那個「島」，是850公里之外的台灣島！環台一圈約1,300公里，挑戰自己是藉口，擁抱一切擦身而過的美麗事物才是正經事！
記者：溫曉嵐　攝影：溫曉嵐、張柏基

環台精采景點逐個捉

台北 → 新竹 → 苗栗 → 台中 → 彰化 → 雲林 → 台南 → 屏東 → 台東 → 花蓮 → 宜蘭

攻略篇：逆時針先甜後苦

對於單車配套不足的香港人來說，台灣簡直是個單車天堂，不少車友從網絡看過台灣車友分享的環島壯麗風景圖輯，紛紛加入環島行列！其實單車環島沒有想像中難，只要有一台基本21段變速的單車，已經可以上坡下坡暢通無阻。時間長短則視乎個人體能：素有鍛煉者，七日環台，日騎約200公里，是為挑戰級；每日平均騎80至100公里，花12至14日完成旅程，則是最常聽到的環島案例，時間足夠騎士欣賞沿途風土人情。這次副刊小記聯同動新聞記者阿仙、攝影師阿基，以及補給車司機雞蛋倫，一行四人從台北出發，目標是12天內完成1,300公里的單車環台之旅！
香港人大都選擇在台北出發，逆時針經桃園、新竹往南部走，經恆春上台東、花蓮、宜蘭完成一圈，此走法的優點是可由城市慢慢進入鄉郊，熟悉台灣的道路後才面對「南迴最高點」壽峠，以及蘇花公路兩大挑戰。順時針走亦未嘗不可，但先苦後甜的玩法只適合進階車友。路途中車友相遇，總會親切問候「你從哪裏來？」，答案除關乎身份認同的「香港」之外，不外乎「走海線」或「走山線」。山線及海線風景迥異，我們這次在西邊以走海線為主，沿途有巨型風車作伴；東邊則走山線，飽覽花東縱谷盛夏的美景。

缺口在恆春

其實環島是個概念，體力不支或遇上惡劣道路情況都不要勉強騎乘，不少台灣人因了解蘇花公路的險峻，都會選乘火車跳過此路段。假如閣下偏執地要環繞台灣本島外圍騎一圈，實質也不可能，皆因環島地圖上還有一個缺口——恆春半島的台26線仍未完工，目前要從南部往東部，走迂迴的台26線須再接上縣道200、199及台9線；直接一點，則可於楓港直接上台9線，或經車城走縣道199再接台9線。

實戰篇：戰前特訓

台灣不少單車店都有環島單車出租，台北市內湖區的港墘路及瑞光路一帶更號稱是「單車一條街」。位於港墘路的「單車家族」沒有幾千呎店面，但老闆張水旺是軍隊少校出身，跟他租車不但裝備齊全，更可要求即時軍訓式單車特訓！老闆從換檔轉波、上斜落山的煞車技巧，以至甩鏈、換胎、補胎的危機應變知識都會全盤教授！以下由水旺老闆為大家講解幾項在台灣騎車的基本安全守則：

一）踩單車要威，最好先戴頭盔。
二）台灣為左軚，公路上單車要靠右騎。

三）在各大城市交通繁忙的十字路口，經常會出現「機、慢車兩段左轉」標誌。單車若要左轉，必須先駛至對面路口的待轉區，待橫向馬路綠燈亮起方可前進。
