火象星座

（白羊座、獅子座、人馬座）

Lilith's Say：

火象星座的人性格比較熱情主動，具備強大動力和意志力，樂於尋找刺激，並喜歡讓自己處於活躍的狀態中。

Meegan's Say：

不妨將這些熱情的色系放在胭脂或唇膏上作聚焦，另外一雙整齊的眉毛及一條明確的眼線，可以突顯火象星座人的意志力。膚色較白的人，可選用古銅色粉底表現外向的個性。

1. 繪畫一條較粗的眉形增加剛強感。

2. 用橙色唇筆畫上眼線後，再沿着眼窩的孤位畫半月形。

3. 以斜掃方式於顴骨位掃上橙色胭脂。

4. 先用唇筆繪畫唇形，再塗上橙色唇膏。

土象星座

（金牛座、處女座、山羊座）

土象星座的人注重現實，個性慢熱及被動，本能地知道如何運用現有的資源去達到實際目標。

2. 於整個眼窩塗上啡色眼影打造輪廓。

3. 繪畫一條啡色眼線。

4. 最後配上橙啡色唇膏，令妝感效果更整體一致。

（巨蟹座、天蠍座、雙魚座）

水象星座的人猶如天使與魔鬼的混合，一方面用經歷來感覺世界，另一方面由於過於敏感特性，令他們易顯得軟弱。

帶有神秘、冶艷感覺的煙熏眼，最適合神秘的水象人。另外黑白分明的眼妝能突顯水象人的獨特個性。

步驟：

1. 於眼窩均勻塗上白色眼影。

2. 用黑色液體眼線筆繪畫一條粗眼線，眼頭及眼尾部份拉長，眼尾部份可微微翹起。

3. 用白色眼線筆繪畫一條內眼線及填滿眼頭的空隙。

4. 最後塗上淺粉紅色唇膏。
